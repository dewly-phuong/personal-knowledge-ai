# Marp

## Description
Marp is a markdown-based slide deck format. It provides a simple and efficient way to create presentations using standard Markdown syntax.

## Usage
Marp is useful for generating presentations directly from structured content, such as a [[wiki/concepts/llm-wiki|wiki]]. The [[wiki/concepts/obsidian|Obsidian]] note-taking application offers a plugin that supports Marp, enabling users to create and display slide decks from their wiki pages.
