# Dataview

## Description
Dataview is an [[wiki/services/obsidian]] plugin that runs queries over page [[wiki/concepts/yaml-frontmatter]] within an Obsidian vault. It is mentioned as a useful tool for enhancing the organization and querying capabilities of a knowledge base built using the [[wiki/concepts/llm-wiki-pattern]].

## Functionality
This plugin enables the generation of dynamic tables and lists by querying the YAML frontmatter added to wiki pages. When an LLM adds metadata such as tags, dates, or source counts to the frontmatter of pages, Dataview can use this information to create live, up-to-date views of the wiki's content. This significantly aids in navigation and information retrieval by allowing users to create custom summaries and indexes based on metadata.
